Bonus 1: Meine erste Webseite (Struktur)

Ziel: Deine digitale Visitenkarte

In diesem Auftrag lernst du, wie du der KI eine einfache Idee (deine Visitenkarte) gibst und sie daraus eine HTML-Webseite baut. Du konzentrierst dich nur auf den Inhalt und die richtige Struktur.

Dein Start-Prompt:

Erstelle eine einfache HTML-Seite für eine persönliche Visitenkarte. Es ist meine allererste Webseite.

Die Seite soll enthalten:
1. Eine grosse Hauptüberschrift mit meinem Namen: [Dein Name].
2. Eine kleinere Überschrift mit meiner Berufsbezeichnung: "Lernender [Dein Beruf]".
3. Einen kurzen Paragrafen über mich: "[Schreib 1-2 Sätze über dich oder deine Hobbys]".
4. Eine Liste mit meinen "Top 3 Lieblings-Dingen" (z.B. Filme, Games, Essen).

So geht's:

  1. Kopiere den Prompt und passe ihn an.
  2. Füge den generierten HTML-Code in eine neue Datei namens `index.html` in VS Code ein.
  3. Speichere die Datei.
  4. Klicke mit der rechten Maustaste auf die Datei in VS Code und wähle "Open in Default Browser".
  5. Fertig! Deine erste Webseite ist live (auf deinem Rechner).

Mögliches Ergebnis:

deine-webseite.html

Nächste Schritte (Iteration):

Geh zurück zur KI und sage: "Füge einen Link zu meiner Lieblings-Webseite hinzu" oder "Mache die Liste nummeriert".
