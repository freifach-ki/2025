Auftrag 11.1: Link teilen

Das Class-Showcase Repository

Damit wir nächste Woche alle Projekte gemeinsam anschauen können, senden Sie mir bitte Ihren Link via Teams.

1

Status prüfen (Öffentlich?)

Für GitHub Projekte: Gehen Sie auf github.com zu Ihrem Repository. Prüfen Sie in den "Settings", ob es auf Public (Öffentlich) gestellt ist.

Für NotebookLM: Klicken Sie oben rechts auf "Teilen" und stellen Sie sicher, dass der Link für "Jeden mit dem Link" zugänglich ist.

2

Link kopieren

Kopieren Sie den/die zutreffenden Link(s):

  • Das GitHub Repository (z.B. `github.com/name/projekt`)
  • Die GitHub Pages Seite (z.B. `name.github.io/projekt`)
  • Das NotebookLM Notebook (Public Link)
3

Via Teams senden

Senden Sie den/die Link(s) einfach per Chatnachricht an Pietro Rossi. Keine Formalitäten nötig, der Link reicht.

Erledigt? Dann zurück an die Arbeit!
