Auftrag 8.2: Klonen wie ein Profi (GitHub Desktop)

Dein Ziel

Niemand programmiert alles von Grund auf. Echte Entwickler/innen "klonen" (kopieren) ständig fremde Projekte, um sie zu nutzen oder daraus zu lernen. Dein Ziel ist es, zwei Beispiel-Projekte auf deinen Laptop zu holen und sie in VS Code zum Laufen zu bringen. Wir nutzen dafür die KI als Assistent.

1 Setup: GitHub Desktop installieren

GitHub Desktop ist ein Programm, das dir das Klonen und Verwalten von Projekten stark vereinfacht. Lass uns das zuerst installieren und einrichten.

Hallo! Du bist ein/e geduldige/r Programmier-Lehrer/in. Ich möchte GitHub Desktop benutzen, um Projekte zu klonen.

1. Gib mir den direkten Download-Link für GitHub Desktop für mein Betriebssystem (Windows oder macOS).
2. Leite mich durch die Installation und die anschliessende Anmeldung mit meinem GitHub-Konto.
3. Zeige mir, wie ich sicherstelle, dass VS Code als Standard-Editor in GitHub Desktop eingestellt ist.

2 Ein Web-Projekt klonen (weektodo)

Jetzt klonen wir ein einfaches Web-Projekt. Das Ziel ist, den Code auf deinem Computer zu haben und die Webseite lokal im Browser zu öffnen.

Ich habe GitHub Desktop installiert und bin angemeldet. Jetzt möchte ich mein erstes Projekt klonen.

Das Projekt ist hier: `https://github.com/manuelernestog/weektodo`

1. Zeige mir, wie ich auf der GitHub-Seite den "Code"-Button finde und "Open with GitHub Desktop" auswähle.
2. Erkläre mir kurz, was beim Klonen passiert (Code wird heruntergeladen).
3. Zeige mir, wie ich das geklonte Projekt direkt aus GitHub Desktop in VS Code öffne.
4. Zeige mir, wie ich die `index.html`-Datei in VS Code finde und sie mit dem Live Server oder direkt im Browser öffne.

3 Ein Python-Projekt klonen (Ascify-Art)

Dieses Projekt ist komplexer. Es braucht zusätzliche Python-Bibliotheken, um zu funktionieren. Hier lernst du den wichtigsten Schritt für Python-Projekte: Abhängigkeiten installieren.

Ich habe das Web-Projekt erfolgreich geklont. Jetzt kommt das Python-Projekt: `https://github.com/Akascape/Ascify-Art`

1. Zeige mir, wie ich auch dieses Projekt klone und in VS Code öffne.
2. Erkläre mir einfach, was die `requirements.txt`-Datei ist und warum sie wichtig ist.
3. Leite mich Schritt-für-Schritt an, wie ich das VS Code Terminal öffne.
4. Gib mir den genauen Befehl, um die Abhängigkeiten zu installieren (z.B. `pip install -r requirements.txt` oder `pip3 ...`) und erkläre, was er tut.
5. Zeige mir, wie ich das Programm gemäss der Anleitung ausführe, um ein Testbild umzuwandeln.

✅ Ziel erreicht!

Du bist fertig, wenn du der Lehrperson zeigen kannst:

  • Die geklonte "weektodo"-App, die in deinem Browser läuft.
  • Das geklonte "Ascify-Art"-Projekt, mit dem du erfolgreich ein Bild im VS Code Terminal umgewandelt hast.
