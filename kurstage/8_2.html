Auftrag 8.2: Klonen wie ein Profi (GitHub Desktop)

Dein Ziel

Niemand programmiert alles von Grund auf. Echte Entwickler/innen "klonen" (kopieren) ständig fremde Projekte, um sie zu nutzen oder daraus zu lernen. Dein Ziel ist es, zwei Beispiel-Projekte auf deinen Laptop zu holen und sie in VS Code zum Laufen zu bringen. Wir nutzen dafür die KI als Assistent.

A Teil 1: Web-Projekt klonen (weektodo)

Du klonst eine fertige To-Do-Liste und siehst dir an, wie ein/e andere/r Entwickler/in das Problem gelöst hat.

B Teil 2: Python-Projekt klonen (Ascify-Art)

Du klonst ein cooles Python-Tool, das Bilder in Text-Kunst umwandelt. Hier lernst du den wichtigsten Schritt: `requirements.txt` installieren.

Dein KI-Assistenz-Prompt:

Nutze diesen Prompt bei deiner KI, um dich durch die Installation und das Klonen führen zu lassen.

Hallo! Du bist ein/e geduldige/r Programmier-Lehrer/in. Ich habe ein GitHub-Konto und VS Code.

Mein Ziel ist es, zu lernen, wie ich fremde Projekte 'klone' und bei mir lokal ausführe. Ich möchte GitHub Desktop dafür benutzen.

Bitte leite mich Schritt für Schritt und gib mir nach jedem Teil einen Meilenstein zur Kontrolle.

**Teil 1: GitHub Desktop einrichten**
1. Führe mich zum Download von GitHub Desktop.
2. Leite mich durch die Installation und die Anmeldung mit meinem GitHub-Konto.
3. Zeige mir, wie ich es mit VS Code verbinde (falls nötig).
(Meilenstein 1: GitHub Desktop ist installiert und ich bin angemeldet.)

**Teil 2: Ein Web-Projekt klonen**
1. Ich möchte dieses Projekt klonen: `https://github.com/manuelernestog/weektodo`
2. Zeige mir, wie ich den "Open with GitHub Desktop"-Knopf auf der GitHub-Seite benutze.
3. Zeige mir, wie ich das geklonte Projekt in VS Code öffne.
4. Zeige mir, wie ich die `index.html`-Datei im Browser öffne.
(Meilenstein 2: Ich sehe die 'weektodo'-Webseite in meinem Browser.)

**Teil 3: Ein Python-Projekt klonen (mit Installation)**
1. Ich möchte dieses Projekt klonen: `https://github.com/Akascape/Ascify-Art`
2. Zeige mir, wie ich es ebenfalls klone und in VS Code öffne.
3. Erkläre mir, was die `requirements.txt`-Datei ist.
4. Leite mich Schritt-für-Schritt, wie ich das VS Code Terminal öffne und den Befehl `pip install -r requirements.txt` (oder `pip3`) ausführe, um die Abhängigkeiten zu installieren.
5. Zeige mir, wie ich das Programm gemäss der `README.md` ausführe (z.B. `python ascify.py -i image.jpg`).
(Meilenstein 3: Ich konnte `ascify.py` erfolgreich ausführen und ein Bild umwandeln.)

Bitte starte mit Schritt 1 von Teil 1.

✅ Ziel erreicht!

Du bist fertig, wenn du der Lehrperson zeigen kannst:

  • Die geklonte "weektodo"-App, die in deinem Browser läuft.
  • Das geklonte "Ascify-Art"-Projekt, mit dem du erfolgreich ein Bild im VS Code Terminal umgewandelt hast.
