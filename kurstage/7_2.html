Projekt 2: Der KI-Witz-Generator (mit API)

Jetzt wird es ernst. Du baust die Witz-Maschine aus Woche 6, aber diesmal holt sie die Witze *live* von einer echten KI. Du lernst, wie man mit `fetch` eine API aufruft – eine der wichtigsten Fähigkeiten im Web-Development.

Dein "Vibe Coding" Prompt:

Hallo KI. Erstelle mir eine komplette "KI-Witz-Generator"-App in EINER `index.html`-Datei.

1.  **Styling:** Benutze TailwindCSS. Die Seite soll zentriert sein.
2.  **HTML-Struktur:**
    * Eine Überschrift "KI-Witz-Generator".
    * Einen grossen Knopf "Neuer Witz!".
    * Einen Lade-Indikator (Spinner), der standardmässig versteckt ist.
    * Einen Absatz mit der ID `witz-text`, wo der Witz erscheinen soll.
3.  **JavaScript-Aktion (API-Call):**
    * Ich habe einen Google API-Key. Zeige mir, wo ich ihn in einer Konstante (`const apiKey = "DEIN_API_KEY_HIER"`) einfügen muss.
    * Wenn ich auf den Knopf klicke:
        a) Zeige den Lade-Spinner an und leere den `witz-text`.
        b) Mache einen `fetch`-Aufruf an die Google Gemini API (`https://generativelanguage.googleapis.com...` mit `gemini-2.5-flash-preview-09-2025`).
        c) Der Prompt an die KI soll sein: "Erzähl einen kurzen, jugendfreien Witz."
        d) Nimm die Text-Antwort der KI, verstecke den Lade-Spinner wieder und zeige den Witz im `witz-text` an.
        e) Füge auch eine Fehlerbehandlung hinzu, falls der API-Aufruf fehlschlägt.

Bitte gib mir den kompletten Code und erkläre ihn in Kommentaren.
