Projekt 2.3: Konsolen-Arcade-Game (Simulation)

Das Projekt

Du baust eine Simulation eines 2D-Arcade-Spiels (wie "Space Invaders" oder "Dodger") komplett im Terminal. Das Spielfeld ist ein 2D-Array (eine Liste von Listen), das in jedem "Frame" (jeder Schleifen-Iteration) neu in die Konsole gezeichnet wird.

Warum dieses Projekt?

Dies ist ein fantastisches Projekt, um die reine **Architektur eines Spiels** zu lernen. Du musst dich nicht mit Grafiken oder komplexen Bibliotheken (wie Pygame) herumschlagen. Du lernst die Kernkonzepte: die **Game Loop** (den "Herzschlag" des Spiels), **State Updates** (Dinge bewegen) und **Collision Detection** (Dinge treffen sich).

Dein "Vibe Coding" Start-Prompt:

Hallo KI. Erstelle ein "Arcade Dodger"-Spiel in EINER Python-Datei (`dodger.py`). Das Spiel soll im Terminal laufen.

1.  **Spielbrett:**
    * Erstelle ein Spielbrett als 2D-Array (Liste von Listen), z.B. 10x10 Felder.
    * Fülle es mit ' ' (Leerzeichen).
2.  **Spieler/in & Gegner:**
    * Platziere den/die Spieler/in ('P') in der Mitte der untersten Zeile.
    * Platziere einen Gegner ('G') in der obersten Zeile an einer zufälligen Position (benutze das `random`-Modul).
3.  **Brett anzeigen:**
    * Eine Funktion, die den Bildschirm löscht (`import os; os.system('cls' if os.name == 'nt' else 'clear')`).
    * Eine Funktion, die das 2D-Array als Raster in der Konsole ausgibt (mit Rändern, z.B. '|' und '-').
4.  **Haupt-Schleife (Game Loop):**
    * Starte eine `while True`-Schleife.
    * In der Schleife:
        a) Zeige das aktuelle Brett an.
        b) Frage den/die Spieler/in nach einem Zug ("a" für links, "d" für rechts).
        c) Aktualisiere die Position von 'P' basierend auf der Eingabe.
        d) **Gegner-Logik:** Lasse den Gegner ('G') eine Zeile nach unten rücken. (Wenn er unten ankommt, setze ihn oben auf eine neue zufällige Position).
        e) **Collision Detection:** Prüfe, ob die Position von 'P' dieselbe ist wie die von 'G'.
        f) Wenn ja: Zeige "Game Over!" an und beende das Spiel (`break`).

Bitte erkläre den Code mit Kommentaren, besonders die Game-Loop und die 2D-Liste.
